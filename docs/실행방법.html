YJHMOCAD 실행 방법

PMSM 기초설계 도구 · 1250W-jk 검증 해석엔진 · 작성일 2026-06-13

1. 준비물 (최초 1회)

  • Node.js 18 이상 — https://nodejs.org 에서 LTS 버전 설치
  • 설치 확인: 터미널(명령 프롬프트 또는 PowerShell)에서 node -v 입력 → 버전이 표시되면 OK

2. 프로그램 실행

  1. 터미널 열기 — 시작 메뉴에서 "PowerShell" 또는 "명령 프롬프트" 실행
  2. 앱 폴더로 이동 
    cd c:\Users\user\Desktop\YJHMOCAD\app
  3. 개발 서버 시작 
    npm run dev
  4. 브라우저로 열기 — 터미널에 아래처럼 주소가 표시됨 
      ➜  Local:   http://localhost:5173/
    이 주소를 Ctrl + 클릭하거나, 브라우저 주소창에 직접 입력
💡 처음 받은 PC이거나 npm run dev에서 모듈 오류가 나면, 같은 폴더에서 npm install을 먼저 한 번 실행한 뒤 다시 npm run dev 하세요.

3. 종료 방법

  • 터미널 창에서 Ctrl + C → 서버 종료
  • 브라우저 탭은 그냥 닫으면 됨

4. 탭 구성 요약

탭기능
GeometryDXF 단면 불러오기 + 파라메트릭 형상 매칭, 측정도구(거리/반경), 레이어 토글
Winding슬롯 단면 실시간 도선 패킹 — 와이어 지름·턴수·가닥수·라이너를 바꾸면 즉시 반영. AWG/Metric/SWG 게이지 테이블, Wedge Model, 슬롯 면적 분해·충전율
Materials강판/자석 재질 선택, 중량 계산, 철손계수 편집
Calculation구동조건(속도·전류·전압·온도), 해석모델 보정계수
Output Data성능 출력 전체 (Motor-CAD FEA 참조값 비교열 포함)
GraphsBEMF/토크/전류 파형, 권선 MMF·공간고조파, 코일 페이저

5. 자주 겪는 문제

증상해결
'npm'은(는) 내부 또는 외부 명령...Node.js 미설치 — nodejs.org에서 설치 후 터미널을 새로 열기
Cannot find module ... 오류app 폴더에서 npm install 실행 후 재시도
5173 포트가 이미 사용 중기존 터미널의 서버를 Ctrl+C로 끄거나, 자동으로 뜨는 5174 등 다른 주소 사용
화면이 깨져 보임브라우저 새로고침(Ctrl+F5). 그래도 안 되면 서버 재시작
입력칸을 지웠더니 "⚠ 입력값 비정상" 표시정상 동작 — 값을 다시 입력하면 자동으로 사라짐 (마지막 유효 결과를 유지해 주는 기능)

6. 참고

  • 설계 데이터는 우측 상단 "설계 JSON 내보내기" 버튼으로 저장 가능
