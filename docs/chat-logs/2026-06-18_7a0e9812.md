## FEMM 철손 직접 적분 → cFe 자동산출 (완료)

FEMM 브릿지가 이제 철손을 직접 계산합니다: **전기 1주기(12스텝) × 치/요크 348점 공간샘플 → Σ Bpk²·mass[kg·T²] 적분**. 앱은 "FEMM 해석 → 보정 적용" 시 `cFe = FEA적분 / 앱첨두근사(mTooth·Bt²+mBy·By²)` 를 **자동 산출**합니다(수동 입력 불필요). compute는 `cal.cFe`를 우선 적용하고, 결과·버튼에 표시됩니다.
