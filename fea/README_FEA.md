# FEMM 자동연동 (FEA 기반 성능)

웹앱의 **"FEMM 해석"** 버튼이 로컬 브릿지 서버를 통해 무료 FEMM으로 진짜 2D FEA를
돌리고, 그 결과(평균토크·토크리플·코깅·에어갭 자속밀도)를 성능으로 가져온다.

## 1회 설치

1. **FEMM** 설치 (https://www.femm.info, 무료) — Windows.
2. **Python** 설치 (https://python.org, 3.9+).
3. 의존성:
   ```
   pip install pyfemm flask flask-cors
   ```

## 실행

1. 브릿지 서버 실행 (FEMM 설치된 PC에서):
   ```
   python fea/femm_server.py
   ```
   → `http://localhost:8765` 에서 대기.
2. 웹앱(`npm run dev`, localhost:5173) → Calculation 탭 → **"FEMM 해석 (FEA)"** 클릭.
3. FEA라 수십 초~수 분 소요. 끝나면 FEMM 기반 성능이 표시됨.

## 동작

- 웹앱이 형상(슬롯/자석 폴리곤)·권선 테이블·전류·진각·병렬회로수를 브릿지로 전송.
- 브릿지(pyfemm)가 FEMM 모델을 구성 → 회전자를 여러 위치로 돌리며 해석:
  - **무부하 1점**(자석 d축을 A상축에 정렬) → 쇄교자속으로 **Ke·역기전력 피크**, 에어갭 자속밀도
  - **부하 토크 스윕**(코깅 1주기, 부하각 고정 동기전류) → 평균토크·토크리플
  - **코깅 스윕**(무전류, 코깅 1주기) → 코깅 p-p
- 결과 JSON 반환 → 웹앱이 해석식 값과 나란히 표시.

## 부하각(토크각) 처리

- 상 자기축 ψ_p = arg(Σ turns·e^{j·pp·φ}) 를 권선 테이블에서 계산하고, 합성 MMF가
  회전자 d축보다 **(90°+진각)** 앞서도록 상전류를 산정한다(I_p = Ipk·cos(Θ−ψ_p)).
  균형 3상이면 상순·부호에 무관하게 합성 MMF가 정확히 목표 방향을 향하므로,
  토크각이 자동으로 MTPA(진각 0) 기준에 맞는다.
- 2D 깊이는 **유효 자기길이(magneticLength)** 를 사용 — 해석식 lam 과 일치(스택길이 사용 시 ~7.5% 과대).

## 속도 / 견고성

- 솔버 정밀도 1e-6, 아크 분할 5°, makeABC 5겹, 스윕은 **코깅 1주기**로 한정 → 위치수·메시 감소.
- `/solve` 는 전역 락으로 직렬화(FEMM 단일 인스턴스 보호). 메시를 거칠게 했으므로
  토폴로지가 1250W-jk 와 크게 다르면 재캘리브레이션 권장.

## 비고 / 한계

- 자기정적 다위치 해석. **철손(Pfe)** 은 추후 추가 예정.
- Ke 쇄교자속은 슬롯 중심 벡터퍼텐셜 가중합(근사) — 해석식 Ke 와 대조해 확인.
- 에러나 토크 부호/크기가 이상하면 서버 콘솔의 trace 와 `psiDeg`(상축 각)를 공유.
